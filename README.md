# Movie-Recommender-System
It is the Web Base Movie Recommender System which Implement Collaborative Filtering By matrix factorizations algorithm.

### Technologies Used

#### Web Technologies
Html , Css , JavaScript , Bootstrap , Django

#### Machine Learning Library In Python3
numpy , pandas , scipy

#### Database
sqlite

##### Requirements
```
python 3.6

pip3

virtualenv
```
##### Setup to run

Extract zip file in your computer

Open terminal/cmd promt

Goto that Path

Example

```
cd ~/Destop/Movie-Recommender-System
```
Create a new virtual environment on that directory

```
virtualenv .
```

Activate Your Virtual Environment

for Linux
```
source bin/activate
```
for Windows
```
cd Scripts/activate
```
Downloading requirements

```
pip install -r requirements.txt
```
It will install all requirements.

### Creating Local Server

goto src directory example

```
cd ../Movie-Recommender-System/src
```
To run
```
python manage.py runserver
```
Now open your browser go to this address
```
  http://127.0.0.1:8000
```
